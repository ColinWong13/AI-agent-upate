## TC-01：爬虫模块

### TC-C01 arXiv 正常爬取

| 项目 | 内容 |
|------|------|
| **编号** | TC-C01 |
| **模块** | arXiv 爬虫 |
| **前置条件** | arXiv API 可正常访问；数据库 papers 表为空 |
| **测试步骤** | 1. 手动触发 arXiv 爬取任务；2. 等待任务执行完毕；3. 查询数据库 papers 表中 source='arxiv' 的记录 |
| **预期结果** | papers 表新增记录；title/authors/abstract/url/pdf_url/published_date/categories 字段非空；crawled_at 为当前时间 |
| **优先级** | P0 |

|------|------|
